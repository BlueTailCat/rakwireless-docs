# 860 ~ 930MHz 3dBi Fiberglass Antenna Datasheet

## Overview

### Features

- Frequency Range: 860 ~ 930&nbsp;MHz
- One antenna to suit both 868 MHz and 915&nbsp;MHz bands 
- Max Gain: 3&nbsp;dBi
- High efficiency 
- Vertically polarized monopole
- Omnidirectional
- Outdoor use
